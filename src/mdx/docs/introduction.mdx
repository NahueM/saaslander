# Introducción

Bienvenido a **TurboSaaS**, tu plantilla SaaS para Next.js + Tailwind CSS, lista para lanzar tu producto en tiempo récord.

## ¿Qué incluye esta plantilla?

- 🌐 **Multi‑idioma** con `next-intl` (`/en` y `/es`).
- 🎨 **Modo claro/oscuro** con `next-themes` y persistencia en `localStorage`.
- 📄 **Documentación in‑app** en MDX, con estilos de Tailwind Typography :contentReference[oaicite:0]{index=0}&#8203;:contentReference[oaicite:1]{index=1}.
- 🚀 **Componentes reutilizables**: Hero, Features, LogoCloud, Pricing, Testimonials, FAQ.
- 💳 **Integración con Stripe** para un flujo de checkout listo para producción.
- ✨ **Animaciones suaves** con Framer Motion.
